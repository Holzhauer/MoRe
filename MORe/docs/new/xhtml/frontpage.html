MORe - Managing Ongoing Relationships

Purpose

The library supports modeling social networks in agent-based models (ABM). A main purpose is to enhance modeling evolving and dynamic networks, which explains the name.

Features

MoRe's features include:

  • Network generators
  • Support for advanced edge objects
  • Calculation and management of network an node measures

Origin

Some of the features and code is based on ReSoNetA, but more directed to Repast Simphony (RS). Like the network facilities in RS and ReSoNetA it is based on the JUNG library. However, it is also intended to be used with other network implementations by building wrappers.
